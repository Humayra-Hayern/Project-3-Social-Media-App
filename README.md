# Sociodora
Attempted to build a MERN full stack social media application where users can create posts and interact with other users' posts.

## Technologies used:

* Javascript
* Express
* NodeJS
* React
* Semantic UI
* GraphQL
* MongoDB Atlas
* Mongoose
